# 小露娜 LUNA — 動畫調參對照表（TUNING）

本機 `npm run dev` 邊看邊調用。**所有捲動特效集中在 `src/scene/scene.js` 的 `render()`**
（約第 101–170 行）。改完存檔，Vite 會即時熱更新。

> 名詞：`p` = 整體捲動進度 0→1（整頁 `#story` 從頂捲到底）。
> 關鍵格格式 `[p, 值]`：到該進度時參數變成該值，中間線性內插。改「值」調強弱、改「p」調時機。

## 2) 參數關鍵格表（`scene.js` 現值）

### `assembly` — 模型炸開↔組裝（0=完全組裝, 1=完全炸開）｜約 L109
| p | 值 | 意義 |
|---|----|------|
| 0.00 | 0.35 | Hero 微散 |
| 0.18 | 0.90 | 開始大散 |
| 0.32 | 1.00 | 完全炸開（散落章最盛）|
| 0.58 | 0.00 | 收合完成（組裝完）|
| 1.00 | 0.00 | 維持組裝 |

- **想讓組裝更晚完成** → 把 `[0.58, 0.0]` 的 `0.58` 調大（如 0.66）。
- **想炸得更開** → `model.js` 的零件 `dist`（見 §4）。

### `spread` — 粒子散落↔收束（1=散, 0=收進核心）｜約 L117
| p | 值 |
|---|----|
| 0.00 | 0.55 |
| 0.30 | 1.00 |
| 0.58 | 0.12 |
| 0.78 | 0.05 |
| 1.00 | 0.00 |

- 與 `assembly` 同步收合，營造「粒子被吸進機身」感。

### `pOpacity` — 粒子透明度（0–1）｜約 L124
| p | 值 |
|---|----|
| 0.00 | 0.50 |
| 0.30 | 0.95 |
| 0.62 | 0.60 |
| 0.72 | 0.90 |
| 0.85 | 0.25 |
| 1.00 | 0.10 |

- 0.72 的小回升=發光時粒子再亮一下；尾段壓低讓品牌字卡乾淨。

### `glow` — 發光強度（0–1，連動光點+Bloom）｜約 L133
| p | 值 |
|---|----|
| 0.50 | 0.00 |
| 0.66 | 1.00 |
| 0.78 | 0.70 |
| 1.00 | 0.55 |

- **光暈更強/更早** → 把峰值 `[0.66,1.0]` 的 p 調小、或值維持 1.0 並見下方 `bloom.strength`。

### `rise` — 能量上升（取代蒸氣，0=無）｜約 L140
| p | 值 |
|---|----|
| 0.60 | 0.00 |
| 0.70 | 0.50 |
| 0.80 | 0.25 |
| 1.00 | 0.00 |

- 只在發光章短暫上升。**想更明顯** → 把 0.70 的值 0.50 調大（如 0.8）。

### `group.rotation.y` — 整體運鏡旋轉（弧度）｜約 L153
| p | 值 |
|---|----|
| 0.00 | −0.5 |
| 0.32 | 0.4 |
| 0.60 | π×0.9 |
| 1.00 | π×1.15 |

- **轉太多頭暈** → 把後段 `π*0.9 / π*1.15` 改小（如 `Math.PI*0.5`）。
- `rotation.x` 與相機 `position.y` 用 `sin(p*π)` 做整段起伏（L159–160）。

### Bloom / 光點 / 相機距離
| 項目 | 現值 | 位置 | 說明 |
|------|------|------|------|
| `bloom.strength` | `0.45 + glow*0.9` | scene.js L163 | 光暈總強度；係數 0.9 調大=發光更炸 |
| `glowLight.intensity` | `glow*3.2` | scene.js L162 | 中央暖光點亮度 |
| 相機 `position.z` | 手機 9.8 / 桌機 8.5 | scene.js `resize()` | 調大=模型更小更遠 |
| `FogExp2` 濃度 | 0.045 | scene.js 建構子 | 調大=背景更暗、景深更重 |

## 5) 常見調整食譜

- **整體更暗更高級** → `FogExp2` 0.045→0.06、`toneMappingExposure` 1.05→0.95。
- **組裝過程更長、更有儀式感** → `assembly` 把 `[0.58,0]` 的 p 往後（0.64），`spread` 同步。
- **發光那一刻更震撼** → `glow` 峰值維持 1.0 + `bloom.strength` 係數 0.9→1.3 + `rise` 0.5→0.8。
- **行動裝置嫌卡** → 粒子 count 降到 ~800、`setPixelRatio` 上限 2→1.5（scene.js 建構子）。
- **想要持續自轉（非只靠捲動）** → 在 `render()` 末段加 `this.group.rotation.y += dt*0.05`。
