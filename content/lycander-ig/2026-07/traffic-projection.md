# LYCANDER IG ｜ 2026 年 7 月流量預估與自動換稿規則（步驟 8）

> ⚠️ **全部為推估，非真實數據**。所有觸及/互動數字皆 `【假設】`，需以帳號真實 IG Insights 校準。
> 本檔目的：① 給每篇一個可被檢驗的 KPI 門檻；② 定義「效果不佳就自行換內容」的規則；③ 示範實際換掉 2 篇。

## 一、預估基準（如何推估，方法透明）

無真實帳號數據，採**相對基準法**：以「帳號近 30 天每篇平均觸及」為 1.0x 基準（記為 `R`），各篇依「形式 × 內容支柱 × 是否壓尖峰時段」給**相對倍率**。
> `R` 待回填真實 Insights（例：若平均觸及 1,200，則 R=1,200）。在此之前所有絕對數字不寫死。

| 形式 | 觸及相對基準【假設】 | 理由 |
|------|----------------------|------|
| 輪播（6 頁） | 1.0–1.3x | 停留時間長、利演算法多次曝光 |
| 單圖懶人卡 | 0.7–0.9x | 停留短，但「存圖」率可能高 |
| Reels | 1.5–3.0x | 觸及天花板最高（非粉絲），變異也最大 |

| 內容支柱 | 互動率傾向【假設】 |
|----------|--------------------|
| 教育/科普 | 儲存率高、留言中 |
| 痛點解法 | 留言/分享高（有共鳴） |
| 社會證明 | 轉換高、互動中 |
| 促銷 | 觸及/互動偏低、導購高（演算法不愛純促銷）|
| 互動徵集 | 留言/分享最高 |

**全月合理預期（若門檻達標）【假設】**：
- 觸及總量約 +20–40%（vs 不發/亂發），來自「集中尖峰時段 + Reels 拉非粉觸及 + 輪播多次曝光」。
- 儲存與分享提升（教育/痛點佔半數）→ 養出後續演算法推薦。
- 導購點擊集中在第 4/7/10 篇。
> 以上為機制推估，**實際以 7 月底 Insights 為準**；達不到就走第三節換稿。

## 三、效果不佳 → 自動換稿規則

**觸發條件（任一）**：
1. 發布後 48 小時互動率 < 該篇門檻的 80%；或
2. 觸及 < 0.6x R（演算法沒推）；或
3. 導購篇（4/7/10）連結點擊 < 門檻。

**換稿動作（依序試）**：
1. **換 hook**：同主題改用 `ad-hook-writer` 的另一框架（如「痛點直擊」換「數據震撼」或「錯誤示範」），只改封面 + caption 首句，重發或改下次同主題用。
2. **換形式**：輪播表現差 → 改 Reels（吃觸及）；促銷觸及被壓 → 包成「教育+軟導購」而非硬促銷。
3. **換時段**：用真實 Insights 重排到實際尖峰（取代 `【需數據】` 基準）。
4. **換主題**：連兩次同支柱都低於門檻 → 該支柱當月降配比，把名額讓給高互動支柱。
- 每次換稿在下方「換稿記錄」留痕（原因 + 改了什麼），回 `data-analyst` 追蹤。

## 四、示範換稿（先換掉 2 篇，落實「自行更換」）

### 換掉 post-07（純促銷，預估觸及 0.8x、易被演算法壓）
- **問題**：硬促銷貼文觸及天花板低，放在 7/15 晚間尖峰「浪費好時段」。
- **換法（換形式＋換 hook，從「促銷」轉「教育型軟導購」）**：
  - 新主題：**「旅充包到底要帶哪幾樣？3 件就夠」** —— 教育骨架，產品自然出現，結尾才軟導購。
  - 新封面 hook（`ad-hook-writer`「結果先行」框架）：『行李箱少一半線，只帶這 3 件』。
  - 形式維持輪播但支柱改「教育/情境」，門檻改 ≥4%（教育標準），導購退到最後一頁。
  - 保留組合連結於 caption 末，不在封面喊折扣。
- **預期**：觸及由 0.8x → 1.1x【假設】，兼顧導購。

### 換掉 post-10（月底又一篇純促銷，與 post-07 支柱重複、易疲乏）
- **問題**：同月第 2 篇硬促銷，受眾促銷疲乏；7/29 尖峰不該再放低觸及型。
- **換法（換主題＋換形式，改成高互動的 UGC/社證收尾）**：
  - 新主題：**「7 月你們的充電包改造成果」** —— 蒐集本月互動篇（post-09）的留言/UGC 做成輪播，社會證明收尾。
  - 新封面 hook（「身份認同」框架）：『如果你也是包包很亂派，這篇給你』。
  - 形式：輪播（精選 UGC）＋ 限動轉發；月底促銷改用**限動**低調帶（不佔貼文尖峰）。
  - 門檻改 ≥4%（社證/互動標準）。
- **預期**：觸及由 0.9x → 1.3x【假設】，並把促銷壓力分散到限動。

> 換稿後 7 月實際支柱配比變為：教育/痛點 6、社會證明 3、互動 1、純促銷 0（促銷改軟導購＋限動）。促銷導購不靠「喊折扣貼文」，靠教育養信任 + 限動收割。
