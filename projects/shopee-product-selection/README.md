### 4.2 競品定位（`competitor-comparison` 方法）
LYCANDER 不打**最低價**，打「**設計一致性 + 安規認證 + 與既有筆電包/鍵盤同生態的搭配感**」。
→ 所有新品**視覺與命名沿用品牌語彙**（磁吸、有序、極細/羽量、認證），讓客人「買包順手把充電、擴充、線一起帶走」，

## 6. 官網（lycander.tw）後台上架 SOP

> 供團隊在後台手動上架；每品約 10–15 分鐘。

1. **建立商品** → 商品名稱貼 §5「商品標題」；分類掛到對應目錄（充電 / 擴充 / 線材 / 配件）。
2. **價格**：售價填 §3 建議售價；如做組合，於「商品選項 / 加購」設定 §5.4。
3. **商品描述**：貼 §5 五點 + 短描述；首圖建議放「情境圖（與筆電包同框）」凸顯生態搭配。
4. **SEO**：把 §5 後台關鍵字填入商品 SEO 標題 / 描述 / 標籤欄位。
5. **庫存 / 物流 / 安規字號**：填實際 SKU、BSMI/認證字號、保固（建議統一 1 年）。
6. **關聯商品**：在每個新品的「推薦 / 一起買」區塊互掛筆電包與彼此，啟動連帶銷售。
7. **同步蝦皮**：官網確認後，把同一份 §5 內容回填蝦皮賣場，維持雙通路一致。
