# EDM 序列產生器（Email Sequence Builder）

這是一個**工具型 skill**。功能：把一個行銷目標（歡迎、棄購挽回、回購…）拆成一條**有節奏、有心理動線**的 email 自動化序列，每封都有明確時機、主旨、內文重點與 CTA。
和 `marketing-team`、`copy-humanizer`、`cx-team` 搭配。

> 原則：序列不是「一直發信」，而是「在對的時機、推進對的下一步」。每封信只負責一個任務。

---

## 何時用

- 要建歡迎序列 / 棄購挽回 / 回購喚回 / 上新預熱
- 既有 EDM 開信率、點擊率差，要重排節奏與主旨
- 新品上市要設計 nurture（培養）到成交的信件動線

## 需要的輸入

1. 序列目標（welcome / 棄購 / 回購 / 上新 / 活動）
2. 受眾階段（新訂閱 / 加購未結 / 已購 N 天 / 沉睡）
3. 產品與主要賣點 / 優惠（若有）
4. 品牌語氣與寄送頻率偏好（可選）

## 常用序列藍圖

- **歡迎序列**：建立信任 → 講品牌故事 → 首購誘因 →（社群/評價）
- **棄購挽回**：提醒 → 排除疑慮（運費/保固）→ 限時誘因
- **回購喚回**：使用建議 → 補充/升級推薦 → 會員專屬優惠
- **上新預熱**：預告 → 痛點教育 → 開賣 → 倒數

## 運作流程

1. **動線設計**：依目標排出每封信的「任務」與心理階段。
2. **時機**：給每封信的觸發條件與間隔（Day 0 / +1 / +3…）。
3. **主旨 × 2**：每封給 A/B 兩個主旨測開信率。
4. **內文骨架**：開頭 hook → 一個重點 → 一個 CTA。
5. **成效指標**：標每封的觀測指標（開信 / 點擊 / 轉換）與調整規則。

| 1 | Day 0 | | | 開信率 |
| 2 | Day +1 | | | 點擊率 |
| 3 | Day +3 | | | 轉換 |

## 各封細節
### 信 1 — <任務>

- 開頭 hook：
- 內文重點（單一）：
- CTA：

### 信 2 …（同上）

## 觸發與分支
- 已開未點 → ？；已點未購 → ？；已購 → 移出序列

## 與團隊串接

- **上游**：賣點 / 痛點由 `review-miner` / `consumers-*` 提供；優惠與毛利空間由 `pricing-calculator` 確認。
- **平行**：文案丟 `copy-humanizer` 去 AI 味；主旨可呼叫 `ad-hook-writer` 取靈感。
- **下游**：序列交 `marketing-team` 設定自動化；售後類序列交 `cx-team`。

> 提醒：優惠 / 保固 / 庫存承諾要與 `operations-team` 對齊，不在信中承諾做不到的事。
