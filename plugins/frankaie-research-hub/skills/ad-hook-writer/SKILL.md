# 廣告 Hook 生成器（Ad Hook Writer）

這是一個**工具型 skill**。功能：把一個賣點 / 痛點，爆量產出**前三秒就抓住人**的 hook，並附短影音開場分鏡。
和 `marketing-team` / `visual-team` 搭配 —— 團隊定方向，這個工具把開頭寫到「滑不走」。

> 原則：hook 不是把賣點講出來，而是製造「我得看下去」的張力。每個 hook 綁一個心理觸發點。

---

## 何時用

- 短影音 / 廣告 / 貼文要寫前三秒開場
- 同一賣點要 A/B 多個開頭測 CTR / 留存
- 既有素材完播率低，要換 hook 救

## 需要的輸入

1. 產品 + 這支素材要主打的**單一**賣點 / 痛點
2. 平台（IG Reels / TikTok / Shopee 短影音 / YouTube）
3. 目標受眾（誰、什麼情境滑到）
4. 語氣偏好（可選：專業 / 親切 / 反差幽默）

## 八種 Hook 框架（每個賣點都跑一輪）

1. **痛點直擊**：點名受眾正在受的苦
2. **好奇缺口**：給一半資訊、逼人看下去
3. **反差／打臉**：推翻常識或對立觀點
4. **數據震撼**：一個違反直覺的數字
5. **身份認同**：「如果你是＿＿，這支必看」
6. **錯誤示範**：「別再這樣＿＿了」
7. **結果先行**：先丟成果再回推方法
8. **情境代入**：把受眾丟進一個熟悉畫面

## 與團隊串接

- **上游**：痛點 / 受眾可由 `consumers-*`（虛擬焦點小組）、`review-miner` 提供。
- **下游**：選定 hook 交 `visual-team` 做分鏡 / 素材；上線後 CTR/留存回 `marketing-team` 迭代。
- **驗證**：產出後可直接丟 `consumers-taiwan/usa/japan/europe` 小組打分，預測哪個 hook 最會被點。

> 提醒：hook 可以有張力，但不得造假承諾或誇大療效 / 規格；涉及功效一律可回溯。
