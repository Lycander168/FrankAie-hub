# SEO 關鍵字擴展器（SEO Keyword Expander）

這是一個**工具型 skill**。功能：給一個產品或種子關鍵字，擴展出**有結構**的關鍵字清單 —— 分主詞 / 流量詞 / 長尾 / 防禦詞，標搜尋意圖與用途，直接餵給 Listing、官網 SEO 或廣告。
和 `ecommerce-operator`、`listing-optimizer`、`marketing-team` 搭配。

> 原則：關鍵字不是越多越好，而是「分對群、放對位置」。每個詞要知道它要去標題、內文、後台還是廣告。

---

## 何時用

- 上架前要備齊 Listing 標題 / 五點 / 後台關鍵字
- 官網 / 部落格要做 SEO 內容規劃
- 廣告要建關鍵字結構（精準 / 廣泛 / 否定）

## 需要的輸入

1. 產品 / 種子關鍵字（中文 / 英文）
2. 目標平台與市場（Amazon US / Shopee TW / Google…）
3. 產品的差異化賣點（用於補語意關鍵字）

## 擴展維度

- **核心詞**：品類主詞
- **規格 / 屬性詞**：功率、介面、材質、尺寸
- **使用情境詞**：旅行、辦公、車用、送禮
- **痛點 / 利益詞**：快充、不發燙、輕巧
- **長尾組合**：核心＋屬性＋情境
- **拼寫 / 同義變體**：含常見錯拼與英美用詞差異
- **防禦 / 品牌詞**：自家與競品品牌詞（廣告用）

## 運作流程

1. **擴展**：依上面維度從種子詞長出候選清單。
2. **分群**：把詞歸到主詞 / 流量 / 長尾 / 防禦四群。
3. **標意圖**：資訊型 / 比較型 / 交易型（決定放內容還是 Listing/廣告）。
4. **標用途**：建議放「標題 / 五點 / A+ / 後台 / 廣告」哪個位置。
5. **優先序**：以相關性與（若有）搜尋量排序；無數據標 `【需查證】`。

## 與團隊串接

- **平行**：清單直接餵 `listing-optimizer`（放對位置）、`ad-hook-writer`（語意靈感）。
- **驗證**：交易型詞的真實搜尋量交 `ecommerce-operator` 用後台 / 工具補。
- **下游**：SEO 主題交 `marketing-team` 排內容行事曆。

> 提醒：搜尋量 / 競爭度無一手後台或工具來源一律標 `【需查證】`，相關性可由語意判斷但需註明非實測數據。
