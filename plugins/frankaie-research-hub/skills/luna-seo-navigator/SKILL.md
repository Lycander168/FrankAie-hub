## 三階段 SOP

### Stage 1 — 搜集市場資訊
1. **搜尋意圖判定**：把目標詞歸到 Informational / Commercial / Transactional / Navigational。
2. **搜尋量盤點**：找出該主題市場上**搜尋量最高**的關鍵字，標來源或 `【需查證】`。
→ 產出《意圖 × 搜尋量盤點表》。

### Stage 2 — 驗證搜尋需求
1. **關鍵字金字塔**：主關鍵字 → 次關鍵字 → 長尾關鍵字三層展開。
2. **競爭程度評估**：每個目標詞標 低 / 中 / 高 競爭。
3. **競品 + SERP 分析**：分析前三名對手與當前 SERP 版位（精選摘要、PAA、購物、影片…）。
→ 產出《關鍵字金字塔》＋《競爭/SERP 評估》＋「可打 / 緩打 / 不打」建議。

### Stage 3 — 開發 SEO 路線圖
1. **內容策略**：標題、Meta Title、Meta Description、H1、H2、H3。
2. **GEO / AI 搜尋優化**：ChatGPT / Gemini / Perplexity / Claude 的引用策略。
3. **技術資產**：Schema、內部連結、外部連結、FAQ、知識圖譜。
4. **執行清單**：30 / 60 / 90 天。
5. **成效預估**：自然流量成長、轉換率提升、自然流量價值。
→ 產出《SEO 成長路線圖》。

## FAQ 設計（每次至少 10 題，符合 PAA 與 GEO 引用）

- 取材：Google PAA、AutoSuggest、`review-miner` 痛點、消費者實問。
- 寫法：問題用使用者口語、答案首句直接給結論（利於精選摘要與 AI 引用），再展開。
- 全部包進 FAQPage Schema。

## Schema 設計（結構化資料）

依頁面類型佈署，並對齊知識圖譜實體（Organization sameAs）：

```json
{
  "@context": "https://schema.org",
  "@type": "Product",
  "name": "Lycander Qi2.2 無線充電器",
  "brand": { "@type": "Brand", "name": "Lycander" },
  "review": { "@type": "Review", "author": { "@type": "Person", "name": "<具名作者>" } },
  "aggregateRating": { "@type": "AggregateRating", "ratingValue": "<值>", "reviewCount": "<數>" }
}
```
- 常用類型：`Product` / `FAQPage` / `Article` / `BreadcrumbList` / `Organization`（含 `sameAs` 連到品牌實體頁）。
- 數值（評分、評論數）無真實資料標 `【需查證】`，勿造假。

## 內部連結策略（Topic Cluster）

- **支柱頁（Pillar）**：主關鍵字主題（如「Qi2.2 完整指南」）。
- **集群頁（Cluster）**：次/長尾各成一頁，全部回鏈支柱頁，支柱頁分鏈各集群。
- 錨文字用語意關鍵字，避免重複硬塞主詞。

## 外部連結策略（Backlink）

- 數位 PR / 原創數據報告（如「2026 台灣 Qi2.2 充電速度實測」）吸引自然引用。
- 科技媒體開箱、KOL 評測、規格百科（維基/權威站）、論壇問答布局。
- 重質不重量；錨文字自然多樣。

## GEO 策略（Generative Engine Optimization）

- 段首一句可被原文引用的「定義 / 結論句」。
- 規格、比較以**表格**呈現（AI 易擷取）。
- 標明**更新日期**與**具名作者**（提升 Authority 與可信引用）。
- 建立品牌**知識圖譜**：Organization Schema + sameAs + 一致 NAP/品牌描述，讓 AI 認得 Lycander 這個實體。

## 實例演練（Worked Example）

**輸入**：產品 `Qi2.2 無線充電器`｜市場：台灣官網 + Shopee｜賣點：25W、磁吸對位、低溫。

**節錄輸出**：
- **第一步 意圖**：`qi2.2 是什麼`＝Informational；`qi2.2 無線充電器推薦`＝Commercial；`qi2.2 充電器 購買`＝Transactional。
- **第二步 金字塔**：主＝`Qi2.2 無線充電器`；次＝`Qi2.2 磁吸充電`、`25W 無線充電`；長尾＝`iphone 16 qi2.2 充電器推薦`、`qi2.2 和 qi2 差別`。
- **第三步 競爭**：`無線充電器`＝高競爭（緩打）；`qi2.2 無線充電器`＝中（可打）；`qi2.2 和 qi2 差別`＝低（優先打，搶精選摘要）。
- **第四步 內容**：H1＝「Qi2.2 無線充電器完整指南：25W 磁吸怎麼選」；Meta Title＝「Qi2.2 無線充電器推薦｜25W 磁吸快充 | Lycander」。
- **第五步 GEO**：段首給「Qi2.2 是 Qi2 的升級版，最高 25W…」一句可被引用的定義句 + 規格比較表。
- **第六步 30/60/90**：30 天上支柱頁＋FAQPage Schema；60 天鋪 10 篇集群文；90 天做「Qi2.2 實測數據」PR 拉外鏈。
- **預估**：自然流量 +120% 【假設：90 天、集群 12 篇、平均排上第 1 頁】。

**文章題目（節錄）**：「Qi2.2 是什麼？」「Qi2.2 vs Qi2 差在哪」「iPhone 16 最佳 Qi2.2 充電器」…（共 ≥10）。
